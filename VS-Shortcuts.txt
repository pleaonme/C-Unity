Код:
--------
Ctrl + K, Ctrl + C                    // Комментировать выделенный фрагмент
Ctrl + K, Ctrl + U                    // Раскомментировать выделенный фрагмент
Ctrl + D                              // Дублировать строку или выделенный фрагмент
Ctrl + X / C / V                      // Вырезать / Копировать / Вставить
Alt + Стрелка вверх/вниз              // Переместить строку вверх/вниз
Ctrl + Shift + V                      // Переключиться по буферу обмена
Ctrl + Пробел                         // Подсказки IntelliSense

Навигация:
-----------
Ctrl + ,                               // Перейти к любому файлу/символу/типу
F12                                    // Перейти к определению
Ctrl + - / Ctrl + Shift + -            // Перейти назад/вперед
Ctrl + F12                             // Перейти к объявлению
Ctrl + Tab                             // Переключение между открытыми вкладками

Сборка/Запуск:
----------
Ctrl + Shift + B                       // Сборка решения
F5                                     // Начать отладку
Ctrl + F5                              // Запустить без отладки
Shift + F5                             // Остановить Отладка

Отладка:
-----------
F9                                     // Переключить точку останова
F10                                    // Шаг с заходом
F11                                    // Шаг с заходом
Shift + F11                            // Шаг с выходом
Ctrl + Alt + I                         // Открыть окно
Ctrl + Alt + L                         // Обозреватель решений
Ctrl + Alt + O                         // Окно вывода

Поиск:
--------
Ctrl + F                               // Найти
Ctrl + H                               // Заменить
F3 / Shift + F3                        // Найти следующий/предыдущий
